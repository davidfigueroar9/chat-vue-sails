Example chat, vuejs and sails
=============================
### Instalations
For install example in the localhost, clone the project and run the bellows comands
For run backend api sails run
```sh
$ cd chat-vue-sails
$ cd backend
$ npm install
$ sails litf
```

For run frontend app vuejs run
```sh
$ cd chat-vue-sails
$ cd frontend
$ npm install
$ npm run dev
```
